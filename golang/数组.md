# 数组

 1. 数组的容量是数组类型的一部分,容量不同的数组不能进行比较

 2. 多为数组定义

    ```go
    //多为数组定义
    //容量为3的 保存容量为2的数组
    
    var a3 [3][2]int
    a3 = [3][2]int{
    		{1, 2},
    		{3, 4},
    		{5, 6},
    	}
    fmt.Println(a3) //[[1 2] [3 4] [5 6]]
    
    ```

    

3. 数组是值类型

   ```go
   a1 := [3]int{1,2,3}
   a2 := a1
   a2[0] = 100
   fmt.Println(a1,a2)  //[1 2 3] [100 2 3]
   ```
